**Задание 1**

Создать Deployment приложения, состоящего из двух контейнеров и обменивающихся данными.

1. Создать Deployment приложения, состоящего из контейнеров busybox и multitool.

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: deployment-emptydir
  labels:
    app: dep-empty
spec:
  replicas: 1
  selector:
    matchLabels:
      app: dep-empty
  template:
    metadata:
      labels:
        app: dep-empty
    spec:
      containers:
      - name: busybox
        image: busybox
        command: ['sh', '-c', 'while true; do echo Hello world! >> /output/output.txt; sleep 5; done']
        volumeMounts:
        - name: emptydir-volume
          mountPath: /output
      - name: multitool
        image: wbitt/network-multitool:latest
        ports:
        - containerPort: 80
        env:
        - name: HTTP_PORT
          value: "80"
        volumeMounts:
        - name: emptydir-volume
          mountPath: /input
      volumes:
      - name: emptydir-volume
        emptyDir: {}
```

**Задание 2**

Создать DaemonSet приложения, которое может прочитать логи ноды.

1. Создать DaemonSet приложения, состоящего из multitool.

```yaml
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: deployment-hostpath
  labels:
    app: dep-hostpath
spec:
  selector:
    matchLabels:
      app: dep-hostpath
  template:
    metadata:
      labels:
        app: dep-hostpath
    spec:
      containers:
      - name: multitool
        image: wbitt/network-multitool
        volumeMounts:
        - name: logs
          mountPath: /input/logs
        ports:
        - containerPort: 80
      volumes:
      - name: logs
        hostPath:
          path: /var/log/syslog
```

2. Обеспечить возможность чтения файла `/var/log/syslog` кластера MicroK8S.

```yaml

          path: /var/log/syslog
```
